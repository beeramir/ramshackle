About Ramshackle

What is this thing?

It's a basic responsive framework with some light styling. It handles the under the hood stuff so you can quickstart a prototype or a simple website in no time. But it might catch on fire. So, there's that.

Who should use this thing?

I'll be straight with you. I started learning web development about 4 months ago. If you're the kind of person who loves lots of unique bugs features then this is going to be awesome for you.

Why did you build this thing?

I'm an aspiring front-end developer that wanted a good project to improve my coding skills, learn responsive design, to have a nice project to add to my portfolio, and a system that I can use to build other sites. That's like 4 birds with 1 stone!

Why do you ask so many things?

Because I need some content to fluff up this section of the site.

Responsive Grid

This grid uses a base-10 system with a max width of 1000px. You can add columns in increments of 10 all the way up to 100. It's like the metric system of responsive grids, but also allows for some in-between sizes like 1/4, 1/3, 2/3, and 3/4. The max width can be changed with a single number, and all content should scale proportionally. Ramshackle also uses 2 different media query breakpoints for tablets and smartphones so your content will look great on any device.

Go ahead and resize the browser, and take it for a spin.

The Syntax



            <div class="grid-container">

            <div class="row">

            <div class="col four">Four</div>

            <div class="col six">Six</div>

            </div>

            <div class="row">

            <div class="col quarter">1/4</div>

            <div class="col three-quarters">3/4</div>

            </div>

            </div>

.grid-container
Acts as the centered wrapper with a max width of 1000px.

.row
All columns should be the immediate children of a row.

.col
Add 'col' and a number and you're all set.

Additional Features

.hero
This is for adding background images that take up 100% of the viewport.

.main-container
This can be used for full width background colors for sections, or for your entire page.

.col-row
Use this if you'd like to convert a column into a row. You can then place multiple columns inside.
Please note that using this will cause your margins to scale.

.flush
Ramshackle adds in padding for each column by default. If you'd like to remove that and tighten things up, add this class to a row, column, or to any other element.

.v-25 and .v-50
This will add 25px or 50px of vertical margin for when an element needs some room to breathe.

.h-25 and .h-50
This will add 25px or 50px of horizontal margin for when an element needs some room to stretch.

.left .right and .center
When applied to a row it will effect the columns within that row horizontally. When applied to a column it will align the content within it.

Built-In Color Palette

Take advantage of the built in color palette system to make styling your website super easy. Customize your colors in a single location in the included style.css, and then use any of the built in classes to control the text or background color.

Buttons

Buttons come in 2 styles. Add the .btn class to an anchor <a> or to a button <button> for a plain button. Or use .btn-fancy to add some style to your links. You can also use the above color palette system to change the text and background colors. Leave it blank for a transparent button.
